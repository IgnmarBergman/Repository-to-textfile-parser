Qu'est ce que ToText :
  ToText est une commande qui prend un Rootfolder et le transforme en fichier text
  contenant une description detaillée du projet ainsi que le contenu de ses fichiers.

Comment utiliser ToText : 
  ToText a été développé sur un Ubuntu donc le system doit etre equivalent, sinon de légers 
  changements peuvent rendre le code fonctionelle sur windows ou MAC.
  xclip doit aussi etre installé sur linux pour que le programme marche puisque ToText a 
  besoin d'accédé au presse papier.

  Si tous c'est critères sont rencontrés, simplement ajouter ToText dans les Path variables.
  maintenant ecrire; ToText dans le terminal pour voir son utilisation avec des exemples.
  
Pourquoi utiliser ToText : 
  ToText a été créé pour simplifié les échanges avec les IA tout en restant anonyme puisque
  il n'est plus mandatoire de faire un compte Github (qui a besoin d'information sur la machine)
  pour partager des structures de dossiers élaborer avec un agent IA.
  Il est donc possible de s'informer auprès de son code avec un IA sur une machine non compromis.

  IMPORTANT! ce code est fait puisque l'anonymat sur internet est moralement important, 
    il n'a pas comme but de servir d'outils pour des pirateur immoral!
